# Customer Churn Prediction Web App

A Machine Learning web application that predicts whether a customer will churn or not using a trained ML model.

Built using **Python, Scikit-Learn, and Streamlit**.

---

## Project Overview

Customer churn prediction helps businesses identify customers who are likely to stop using their service.

This application allows users to input customer details and predicts whether the customer is likely to churn.

## Installation

Clone the repository

```bash
git clone https://github.com/yourusername/customer-churn-prediction.git
```

Go to project folder

```bash
cd customer-churn-prediction
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the app

```bash
streamlit run app.py
```
